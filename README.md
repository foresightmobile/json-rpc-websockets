# JSON-RPC WebSockets

JSON-RPC WebSockets is an open-source implementation of [JSON-RPC 2.0](https://www.jsonrpc.org/specification) using WebSockets.

Inspired by [rpc-websockets](https://www.npmjs.com/package/rpc-websockets).
